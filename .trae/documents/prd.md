## 1. 产品概述

一个基于阿里云 ESA Pages 部署的视频播放网站，视频资源存储于七牛云 OSS，使用 Video.js v8 作为核心播放器。用户可以浏览视频列表、查看视频详情并选择播放不同视频。

- 目标用户：需要在线观看视频内容的普通用户
- 核心价值：提供流畅、美观的视频浏览与播放体验

## 2. 核心功能

### 2.1 用户角色

无需角色区分，所有访客均可浏览和播放视频。

### 2.2 功能模块

1. **视频列表页**：视频封面网格展示、搜索筛选、分类浏览
2. **视频播放页**：Video.js 播放器、视频信息展示、相关视频推荐

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 视频列表页 | 顶部导航栏 | 网站标题、搜索框 |
| 视频列表页 | 视频卡片网格 | 展示视频封面、标题、时长，点击进入播放页 |
| 视频列表页 | 搜索筛选 | 按标题关键词搜索视频 |
| 视频播放页 | 视频播放器 | Video.js v8 播放器，支持播放控制、全屏、音量调节 |
| 视频播放页 | 视频信息区 | 视频标题、描述展示 |
| 视频播放页 | 相关视频推荐 | 侧边栏展示其他视频，点击切换播放 |

## 3. 核心流程

用户打开网站 → 浏览视频列表 → 点击感兴趣的视频卡片 → 进入播放页 → Video.js 加载并播放视频 → 浏览侧边推荐列表 → 点击切换其他视频

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深色影院风格（#0a0a0f 深黑背景 + #e50914 红色强调色）
- 辅助色：#1a1a2e 深蓝灰卡片背景、#ffffff 白色文字
- 按钮风格：圆角微凸，hover 时发光效果
- 字体：标题使用 Outfit（粗体、大号），正文使用 DM Sans
- 布局风格：卡片网格布局，播放页左右分栏
- 图标风格：线性图标，简洁现代

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 视频列表页 | 顶部导航栏 | 深色半透明背景、网站 Logo 左侧、搜索框右侧、毛玻璃效果 |
| 视频列表页 | 视频卡片网格 | 3列网格、封面图 16:9 比例、底部渐变遮罩显示标题和时长、hover 放大+阴影效果 |
| 视频播放页 | 视频播放器 | 居中大尺寸播放器、深色背景、Video.js 自定义深色主题 |
| 视频播放页 | 视频信息区 | 播放器下方、标题大号粗体、描述灰色小字 |
| 视频播放页 | 相关视频推荐 | 右侧竖向列表、小封面+标题+时长、hover 高亮 |

### 4.3 响应式设计

- 桌面端优先设计（1920px 基准）
- 平板端（768px-1024px）：视频卡片 2 列，播放页推荐列表折叠到底部
- 移动端（<768px）：视频卡片单列，播放页全宽播放器，推荐列表水平滚动
